# Acerca de

Stackyard es un proyecto personal de código abierto: un gestor de entornos
de bases de datos local basado en Docker y un cliente de BD multi-motor,
creado para reemplazar la rutina habitual de "escribir a mano un
docker-compose.yml y luego abrir un cliente GUI separado" con una única
app nativa.

Se distribuye bajo la
[PolyForm Noncommercial License 1.0.0](https://polyformproject.org/licenses/noncommercial/1.0.0) —
se permite el uso personal y no comercial, con atribución requerida.
Ver [`LICENSE`](https://github.com/KamerrEzz/stackyard/blob/main/LICENSE)
para los términos exactos. No es un producto comercial: no hay un plan de
pago, ninguna empresa detrás, ni planes de venderlo.

## Cómo fue construido

Stackyard fue construido mediante una colaboración estrecha entre un
desarrollador humano, que dirigió la arquitectura, las decisiones de
producto y el alcance en cada etapa, y Claude (Anthropic), que ejecutó el
trabajo de implementación, pruebas y verificación bajo esa dirección. El
esfuerzo se repartió de forma aproximadamente equitativa entre ambas
partes de ese proceso: el humano tomó las decisiones sobre qué construir
y por qué; la IA escribió y verificó una gran parte del código resultante.
Ninguna de las partes trabajó sin supervisión: cada fase en la historia
del proyecto fue revisada y confirmada por el desarrollador humano antes
de avanzar a la siguiente.

Esta divulgación aplica específicamente a esta página. No cambia cómo se
escribe, documenta o acredita el resto del proyecto en ningún otro lugar.
